        Colophon

        A colophon is the note at the back of an old book that tells you how it was made — the type, the press, the hands involved. Here is this one's.

        The look

        This page is dressed like an antique letterpress book: warm ivory paper in place of glaring white, ink-black text, a single oxblood accent, and serif type with the ink-spread of old metal. Drop caps open the longer pieces, stars divide the sections, and a pointing hand sends you onward. Nothing glows, nothing blinks, nothing competes for your eye — it is meant to feel found, like a quiet book pulled off a shelf. The aim is calm, neutral, and welcoming: a place to read slowly, not a thing to be optimised.

        Typefaces

        Set throughout in IM Fell DW Pica — a digitisation of a 17th-century letterpress face, ink-spread and all — doing every job here: title, headings (in its small-caps cut), and reading text. Served from Google Fonts. The drop caps are Royal Initialen, an ornate old initials face hosted in the repository itself.

        How it's built

        One hand-written index.html. No framework, no build step, no bundler, no dependencies — all the CSS and JavaScript live in the page itself. The whole site is a single file you could read top to bottom.

        Where it lives

        Hosted on GitHub Pages. A git push publishes it; there is no server to keep running.

        How it's written

        New posts are written from a hidden editor inside the site, which commits the change straight to the repository through the GitHub API. The token that allows this is encrypted in the browser with the Web Crypto API and unlocked by a passphrase — no secrets sit in the open, and there is no backend.

        Ornaments

        Drop caps, star dividers, and the pointing hand (☞, a manicule) are old printer's conventions. Engravings, where they appear, come from the public domain.

        In the spirit of

        The small, weird, hand-made web — and in particular From Jason and near.blog, which showed that a personal site can just be calm and yours.
